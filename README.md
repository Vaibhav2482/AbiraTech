# DevCraft Solutions - Modern Premium IT Company Website

A modern, premium IT company website built with React 19, Vite, Material UI, and Framer Motion. Inspired by the FLOQ design style with minimalistic aesthetics, large typography, and smooth animations.

## 🎨 Design Features

- **Minimalistic & Premium** - Clean, elegant design with lots of whitespace
- **Modern Agency Look** - Bold typography and professional layout
- **Responsive Design** - Fully optimized for desktop, tablet, and mobile
- **Smooth Animations** - Framer Motion for elegant transitions
- **Color Scheme** - Light gray background (#F5F5F5), black text (#111111), orange accent (#FF6B3D)

## 📋 Project Structure

```
src/
├── components/          # Reusable React components
│   ├── Header.jsx       # Sticky navigation bar
│   ├── Hero.jsx         # Hero section with CTA
│   ├── ServicesMarquee.jsx  # Auto-scrolling services
│   ├── StatsBar.jsx     # Company statistics
│   ├── FeaturedProjects.jsx # Project portfolio
│   ├── AboutSection.jsx # About us section
│   ├── ProcessTimeline.jsx  # Development process
│   ├── TechnologiesSection.jsx  # Tech stack
│   ├── TestimonialsSection.jsx  # Client testimonials
│   ├── ContactCTA.jsx   # Contact call-to-action
│   └── Footer.jsx       # Footer with links
├── styles/              # Global styles
│   ├── globals.css      # Global CSS
│   └── theme.js         # Material UI theme
├── App.jsx              # Main app component
└── main.jsx             # React entry point
```

## 🚀 Getting Started

### Prerequisites
- Node.js 16+
- npm or yarn

### Installation

1. **Install dependencies:**
```bash
npm install
```

2. **Start development server:**
```bash
npm run dev
```

The site will automatically open at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

Output will be in the `dist/` directory.

### Preview Production Build

```bash
npm run preview
```

## 📝 Customization

### Update Company Info
Edit the following files:
- `src/components/Header.jsx` - Company name and navigation
- `src/components/Hero.jsx` - Headline and subheading
- `src/components/FeaturedProjects.jsx` - Project information
- `src/components/Footer.jsx` - Contact information

### Modify Colors
Update color values in:
- `src/styles/globals.css` - CSS variables
- `src/styles/theme.js` - Material UI theme
